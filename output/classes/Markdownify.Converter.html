fixInlineElementSpacing()

fixInlineElementSpacing()

Moves leading/trailing whitespace from inline elements outside of the element. This is to fix cases like `<strong> Text</strong>`, which if converted to `** strong**` would be incorrect Markdown.

Examples:

  • leading: <strong> Text</strong> becomes <strong>Text</strong>
  • trailing: <strong>Text </strong> becomes <strong>Text</strong>
